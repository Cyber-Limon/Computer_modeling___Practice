###### **--- Дано / Найти ---**

**Дано:**  - множество городов **C = {c\_1, c\_2, ..., c\_n}**;

       - расстояния **d(c\_i, c\_j) ∈ Z+**, **(c\_i, c\_j) ∈ C**.

**Найти:** окрестность **N**.



###### **--- Вход / Выход ---**

**Вход:**  расстояния **d(c\_i, c\_j) ∈ Z+**, **(c\_i, c\_j) ∈ C**.

**Выход:** окрестность **N**.







###### **--- CitySwap ---**

1\. Выбрать начальное решение **S = (c\_1, c\_2, ..., c\_n)**;

2\. **i = 0** // *первая позиция*;

3\. Пока **i < n**:

 		3.1. **j = i + 1** // *вторая позиция*;

 		3.2. Пока **j < n**:

 				3.2.1. **S' = S** // *копирование начального решения*;

 				3.2.2. Меняем местами в **S'** элементы на позициях **i** и **j**;

 				3.2.3. Добавляем **S'** в **N**;

 				3.2.4. **j = j + 1**;

 		3.3. **i = i + 1**.







###### **--- 2-opt ---**

1\. Выбрать начальное решение **S = (c\_1, c\_2, ..., c\_n)**;

2\. **i = 0** // *первая позиция*;

3\. Пока **i < n - 2**:

 		3.1. **j = i + 2** // *вторая позиция*;

 		3.2. Пока **j < n - 1**:

 				3.2.1. **S' = S** // *копирование начального решения*;

 				3.2.2. Разворачиваем в **S'** элементы с позиции **i + 1** до позиции **j** // *включительно*;

 				3.2.3. Добавляем **S'** в **N**;

 				3.2.4. **j = j + 1**;

 		3.3. Если **i != 0**:

 				3.3.1. **S' = S** // *копирование начального решения*;

 				3.3.2. Разворачиваем в **S'** элементы с позиции **i + 1** до позиции **n** // *не включительно*;

 				3.3.3. Добавляем **S'** в **N**;

 		3.4. **i = i + 1**.







###### **--- Сдвиг ---**

1\. Выбрать начальное решение **S = (c\_1, c\_2, ..., c\_n)**;

2\. **k = 1** // *ширина сдвига*;

3\. Пока **k < n**:

 		3.1. **i = 0** // *начальная позиция*;

 		3.2. Пока **i < n**:

 				3.2.1. **j = (i + k) mod n** // *вычисление новой позиции*;

 				3.2.2. **S' = S** // *копирование начального решения*;

 				3.2.3. Сдвигаем в **S'** элемент с позиции **i** на позицию **j**;

 				3.2.4. Если **S'** нет в **N**:

 						3.2.4.1. Добавляем **S'** в **N**;

 				3.2.5. **i = i + 1**;

 		3.3. **k = k + 1**.
